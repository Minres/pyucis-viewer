Running PyUCIS ViewerÔÉÅ

In its current form, PyUCIS Viewer is extremely simple. In order to run the tool, you must have an XML file formatted according to the UCIS XML interchange format.

A common way to obtain such a file is by saving collected coverage from the PyVSC library. See the PyVSC Documentation for more details.

PyUCIS Viewer accepts a single argument: the coverage XML file.

% pyucis-viewer cov.xml
